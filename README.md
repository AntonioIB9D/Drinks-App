# Drinks-App 🍹

Buscador de bebidas ✨

La aplicación permite: 

1. Buscar bebidas en base a una categoría y un ingrediente. 🔍

---

### Notas adicionales 📗

##### Para utlizar la aplicación se deberan:

---

Instalar modulos de node ⬇️ :

```
npm install
```

Construir ejecutar ⚙️ :

```
npm run dev
```
